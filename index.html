Projects

Interactive Navigation Menu

This Navigation Menu project is a responsive and interactive menu built using HTML, CSS, and JavaScript that provides seamless navigation across different sections of a website. The menu features a modern, user-friendly design with dropdown options and smooth animations for an enhanced user experience. CSS is used for styling and ensuring the layout adapts well to various screen sizes, making the menu functional on both desktop and mobile devices. JavaScript enables dynamic interactions, such as opening and closing dropdowns or toggling the menu on smaller screens. This project demonstrates the ability to create intuitive and aesthetically pleasing navigation solutions for web applications.

Tic Tac Toe

This Tic Tac Toe web project is an interactive game built using HTML, CSS, and JavaScript that allows users to play the classic game against another player or a simple AI. The game features a clean, responsive design, ensuring a smooth experience on both desktop and mobile devices. JavaScript manages the game logic, including turn-taking, checking for win conditions, and handling draws, while CSS provides a visually appealing layout with animations to enhance user engagement. The project showcases the ability to create intuitive user interfaces and implement basic game algorithms, making it a fun and educational web development exercise..
